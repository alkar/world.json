world.json
==========

A compilation of information for countries in JSON form.

Format:
``` json
{
  "code": "US",
  "name": "United States",
  "capital": "Washington",
  "continent": "NA",
  "currency": {
    "code": "USD",
    "name": "America (United States of America) Dollars",
    "unicode": "0024"
  },
  "languages": [
    "en-US",
    "es-US"
  ],
  "phone_code": "1",
  "zip": {
    "format": "#####-####",
    "regex": "^\\d{5}(-\\d{4})?$"
  },
  "tld": ".us"
}
```

Feel free to download and use, check the [website](http://world.laircorp.org/) for ways to contribute.

Cheers!
